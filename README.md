# Personal Forex Trader: Execution Engine
*A private, AI-assisted trade execution bridge for Windows 11.*

This repository serves as the development hub for a proprietary algorithmic execution engine. The software is designed to bridge local behavioral analysis formulas with a high-performance execution layer.

## 🚀 Key Features
* **Automated Order Execution:** Direct-to-market order placement based on local behavioral triggers.
* **Dynamic Pair Management:** Real-time monitoring and active management of multiple currency pairs simultaneously.
* **Risk Governance:** Automated enforcement of Stop-Loss (SL) and Take-Profit (TP) parameters to ensure capital protection.
* **AI-Integrated Development:** Built and maintained using a Claude Code workflow for high-standard code architecture and version stability.

## 🛠️ System Architecture
* **Environment:** Native Windows 11 application logic.
* **Connectivity:** Integration-ready for cTrader OpenAPI v2 (Execution Layer).
* **Data Handling:** Optimized for low-latency tick data processing and local storage.

## 🔒 Security & Privacy
This repository contains only the documentation and architectural framework. 
* **Proprietary Logic:** All mathematical formulas and core strategy files are stored locally and are not part of this public repository.
* **Credentials:** All API keys (`Client ID`, `Secret`) and tokens are managed via local `.env` files and are strictly excluded from version control.
